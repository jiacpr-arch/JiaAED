# แก้ปมชื่อเว็บไซต์ในใบอนุญาต ฆพ.: www.jia1669.com vs jiaaed.com (17 ก.ค. 2026)

ใบอนุญาตโฆษณาทั้งสองใบที่ได้รับ (ฆพ.2475/2569 — Y0/Y2, ฆพ.287/2567 — Y8) ระบุสื่อว่า
**"เว็บไซต์ ชื่อ www.jia1669.com"** แต่เว็บที่ใช้งานขายจริงคือ **jiaaed.com** — สองโดเมนนี้
เป็นเครือเดียวกัน (บ.เจี่ยรักษา จำกัด) แต่ในทางกฎหมาย ต้องทำให้ตรงกับที่ อย. อนุมัติไว้

**ตรวจสอบแล้ว (17 ก.ค. 2026):**
- `jia1669.com` (apex) มีการจดทะเบียนแล้วและชี้ไปโฮสต์ภายนอก (ไม่ใช่ Vercel) — ไม่ต้องซื้อใหม่
- ใน Vercel team เดียวกันมีโปรเจกต์ชื่อ `jia1669` แยกต่างหาก ใช้โดเมน **`subaed.jia1669.com`**
  (คนละส่วนกับ apex/www) — งานนี้**ไม่แตะ**โปรเจกต์หรือโดเมนย่อยนั้นเลย

---

## ส่วนที่ทำในโค้ดแล้ว (เสร็จ ไม่ต้องทำอะไรเพิ่ม)

`middleware.ts` เพิ่ม redirect ระดับโดเมน: ถ้ามี request เข้ามาด้วย Host เป็น
`jia1669.com` หรือ `www.jia1669.com` → 301 redirect ไป `https://jiaaed.com` (คง path เดิม)
โค้ดนี้ **ยังไม่มีผลใด ๆ กับเว็บปัจจุบัน** จนกว่าจะทำขั้นตอน DNS ด้านล่างเสร็จ (deploy ไว้รอเฉย ๆ)

---

## ส่วนที่เจ้าของ/ทีมต้องทำเอง (ผมเข้าไม่ถึง 2 เรื่องนี้)

### A) DNS ของ jia1669.com → ใครก็ตามที่ถือบัญชี registrar ของ jia1669.com

1. เข้าไปที่ผู้ให้บริการโดเมนของ **jia1669.com** (ไม่ใช่ Z.com ที่ใช้กับ jiaaed.com —
   ต้องเช็คว่า jia1669.com จดทะเบียน/DNS อยู่ที่ไหน)
2. เพิ่มโดเมน `jia1669.com` และ `www.jia1669.com` เข้าไปใน Vercel project **jiaaed**
   (Vercel Dashboard → jiaaed → Settings → Domains → Add) — โปรเจกต์นี้คนละตัวกับ
   โปรเจกต์ `jia1669` เดิมที่ถือ `subaed.jia1669.com` อยู่ ไม่กระทบกัน
3. ตั้งค่า DNS records ตามที่ Vercel แจ้ง (โดยทั่วไป A record ของ apex ชี้ Vercel IP +
   CNAME ของ www ชี้ `cname.vercel-dns.com`)
4. รอ propagate แล้วทดสอบ: เปิด `https://jia1669.com` และ `https://www.jia1669.com`
   ต้อง redirect ไป `https://jiaaed.com` โดยอัตโนมัติ (โค้ด middleware ทำให้แล้ว)

### B) แจ้งแก้ไขชื่อสื่อกับ อย. — ทางที่ปลอดภัยที่สุดตามตัวอักษรใบอนุญาต

แม้ทำ redirect แล้ว หน้าจอผู้บริโภคจะเปลี่ยน URL เป็น jiaaed.com ทันที ซึ่งอาจตีความว่า
สื่อที่แสดงจริงไม่ตรงกับที่ขออนุญาตไว้เป๊ะ ๆ — จึงควรยื่นแจ้งแก้ไขข้อมูลสื่อควบคู่กันไป
(เป็นนิติบุคคลเดียวกัน ปกติเป็นการแจ้งแก้ไขข้อมูล ไม่ใช่ขอใบอนุญาตใหม่)

ร่างหนังสือด้านล่างพร้อมให้เจ้าของนำไปพิมพ์หัวจดหมาย บ.เจี่ยรักษา จำกัด ลงนาม
(นางสาว วนิดา แซ่เจี่ย ตามที่ปรากฏในใบอนุญาตทั้งสองฉบับ) แล้วยื่นที่สำนักงาน
คณะกรรมการอาหารและยา (กองผลิตภัณฑ์เครื่องมือแพทย์ / จุดยื่นใบอนุญาตโฆษณา):

```
วันที่ ____________________

เรื่อง  ขอแจ้งแก้ไขชื่อสื่อโฆษณา (เว็บไซต์) ในใบอนุญาตโฆษณาเครื่องมือแพทย์

เรียน  เลขาธิการคณะกรรมการอาหารและยา

        ตามที่บริษัท เจี่ยรักษา จำกัด ได้รับใบอนุญาตโฆษณาเครื่องมือแพทย์ ดังนี้
        1. ใบอนุญาตเลขที่ ฆพ.2475/2569 (เครื่องกระตุกหัวใจไฟฟ้าชนิดอัตโนมัติ
           HeartSave Y0 และ Y2) ออกให้ ณ วันที่ 17 กรกฎาคม 2569
        2. ใบอนุญาตเลขที่ ฆพ.287/2567 (เครื่องกระตุกหัวใจไฟฟ้าชนิดอัตโนมัติ
           HeartSave Y8)
        ซึ่งทั้งสองฉบับระบุสื่อเป็น "เว็บไซต์ ชื่อ www.jia1669.com"

        บริษัทฯ ขอแจ้งว่าเว็บไซต์หลักที่ใช้เผยแพร่โฆษณาผลิตภัณฑ์ดังกล่าวในปัจจุบัน
        คือ www.jiaaed.com ซึ่งเป็นเว็บไซต์ในเครือเดียวกันกับ www.jia1669.com
        (ผู้ประกอบการรายเดียวกัน คือ บริษัท เจี่ยรักษา จำกัด) โดยเนื้อหาโฆษณา
        เป็นไปตามข้อความที่ได้รับอนุมัติแนบท้ายใบอนุญาตทั้งสองฉบับทุกประการ

        จึงเรียนมาเพื่อขอแจ้งแก้ไข/เพิ่มเติมชื่อสื่อเว็บไซต์ในใบอนุญาตทั้งสองฉบับ
        ข้างต้น จาก www.jia1669.com เป็น www.jiaaed.com (หรือระบุทั้งสองชื่อ
        เนื่องจาก www.jia1669.com จะทำการเปลี่ยนเส้นทาง (redirect) มายัง
        www.jiaaed.com โดยอัตโนมัติ) ทั้งนี้ บริษัทฯ ยินดีให้ข้อมูลเพิ่มเติมหรือ
        ดำเนินการตามขั้นตอนที่สำนักงานฯ กำหนดต่อไป

        จึงเรียนมาเพื่อโปรดพิจารณา

                                    ขอแสดงความนับถือ

                            _____________________________
                              (นางสาว วนิดา แซ่เจี่ย)
                                บริษัท เจี่ยรักษา จำกัด
```

**เอกสารแนบที่ควรเตรียมไปด้วย:** สำเนาใบอนุญาต ฆพ. ทั้งสองฉบับ, สำเนาบัตรประชาชนผู้ลงนาม,
หนังสือรับรองบริษัท (ถ้าเจ้าหน้าที่ขอ), print screenshot หน้า jiaaed.com ที่มีเนื้อหาตรงกับ
ข้อความที่ อย. อนุมัติ
