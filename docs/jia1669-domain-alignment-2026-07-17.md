# แก้ปมชื่อเว็บไซต์ในใบอนุญาต ฆพ.: www.jia1669.com vs jiaaed.com (17–19 ก.ค. 2026)

ใบอนุญาตโฆษณาทั้งสองใบที่ได้รับ (ฆพ.2475/2569 — Y0/Y2, ฆพ.287/2567 — Y8) ระบุสื่อว่า
**"เว็บไซต์ ชื่อ www.jia1669.com"** แต่เว็บที่ใช้งานขายจริงคือ **jiaaed.com** — สองโดเมนนี้
เป็นเครือเดียวกัน (บ.เจี่ยรักษา จำกัด) แต่ในทางกฎหมาย ต้องทำให้ตรงกับที่ อย. อนุมัติไว้

**ตรวจสอบแล้ว (17 ก.ค. 2026):**
- `jia1669.com` (apex) มีการจดทะเบียนแล้วและชี้ไปโฮสต์ภายนอก (ไม่ใช่ Vercel) — ไม่ต้องซื้อใหม่
- ใน Vercel team เดียวกันมีโปรเจกต์ชื่อ `jia1669` แยกต่างหาก ใช้โดเมน **`subaed.jia1669.com`**
  (คนละส่วนกับ apex/www) — งานนี้**ไม่แตะ**โปรเจกต์หรือโดเมนย่อยนั้นเลย

---

## อัปเดต 19 ก.ค. 2026 — เปลี่ยนแนวทาง: ใช้ jia1669.com เป็นเว็บจริง แทนการ redirect

แนวทางเดิม (301 redirect jia1669.com → jiaaed.com ผ่าน `middleware.ts`) **ถูกถอดออกแล้ว**
เพราะเจ้าของกำลังตั้งค่า **jia1669.com บนระบบ MakeWebEasy** ให้เป็นเว็บของตัวเอง
พร้อมฝังหน้า `/aed/yuwell-y2` (ราคา ฿59,999) เป็น **iframe** อยู่ในนั้น (ดู `next.config.ts` —
เปิด `frame-ancestors` เฉพาะ jia1669.com/www.jia1669.com ให้แล้วใน PR #121)

**ผลดีทางกฎหมาย:** แนวทางนี้ตรงกับสิ่งที่ ฆพ. ทั้งสองใบระบุไว้ตรง ๆ กว่าเดิม — **www.jia1669.com
จะแสดงเนื้อหาโฆษณาจริง** (ผ่าน iframe) แทนที่จะ redirect หนีไปโดเมนอื่นทันที จึงไม่ต้อง
ชี้ DNS jia1669.com มาที่ Vercel และไม่ต้องยื่นแก้ไขชื่อสื่อกับ อย. เป็นการด่วน — เนื้อหาที่
ปรากฏบน www.jia1669.com คือของจริงตามที่ขออนุญาตไว้

**ยังคงแนะนำเก็บร่างหนังสือแจ้ง อย. ไว้ (ด้านล่าง) เป็นทางเลือกสำรอง** เผื่อในอนาคตต้องการ
โฆษณาชี้ไปที่ jiaaed.com โดยตรงด้วย (เช่น Google/Facebook Ads ที่ final URL เป็น jiaaed.com)
กรณีนั้นควรแจ้ง อย. ให้ครอบคลุมทั้งสองชื่อโดเมนเพื่อความรอบคอบ

## ส่วนที่ทำในโค้ด

`middleware.ts` — **ไม่มี** redirect ระดับโดเมนอีกต่อไป (ถอดออกแล้ว) เว็บ jia1669.com
ยังคงโฮสต์อยู่บน MakeWebEasy ตามเดิม ไม่ถูกดึงมาที่ Vercel

`next.config.ts` — เปิด CSP `frame-ancestors` ให้ jia1669.com/www.jia1669.com ฝัง
`/aed/yuwell-y2` เป็น iframe ได้ (PR #121)
