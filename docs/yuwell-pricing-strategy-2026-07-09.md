## 🧠 Insight ที่เป็นหัวใจของกลยุทธ์

**Y0 กับ Y8 = ฮาร์ดแวร์ตัวเดียวกัน ต่างแค่ "แผ่นแปะ" (มี/ไม่มี Feedback Sensor)**
ที่ผ่านมาขาย Y8 ที่ ฿39,999 = เท่ากับ **แจกฟีเจอร์ CPR feedback ฟรี** ไม่คุ้มกับต้นทุนของที่ต่างกัน

กลยุทธ์ใหม่ = แยกมูลค่าเป็นชั้น ๆ ให้ชัด:
1. **ขายฮาร์ดแวร์เปล่า (Y0) ราคาถูก** เป็นตัวล่อ/ตัวโฆษณา
2. **เก็บค่า CPR feedback ที่ Y8** (ขยับขึ้นเป็นขั้นกลาง)
3. **เก็บพรีเมียมก้อนใหญ่ที่ "จอ EKG" (Y2)** — สิ่งที่คู่แข่งคิดแพงมาก (ZOLL/Mindray รุ่นมีจอ ฿72,500–89,000) แต่เราขายครึ่งราคา

> **หมายเหตุจัดซื้อ:** Y8 ยกล็อต 100 เครื่อง (ทุน ฿23,000) ถูกกว่า Y0 ล็อต 10 เครื่อง (ทุน ฿25,000) ทั้งที่ฟีเจอร์เยอะกว่า → ควรสต็อกตัวถังหลักจากล็อต Y8 ให้มาก แล้วเลือกติดแผ่นแปะตามรุ่นที่ต้องส่ง

## 🟢 เหตุผลรายรุ่น

### Y0 = ฿39,999 — ตัวขายหลัก + ตัวโฆษณา
- ยึด **หมุดราคาที่พิสูจน์แล้วว่าขายดี** (Y8 เดิมขายดีที่ ฿39,999) มาใช้กับ Y0
- ต่ำกว่าโปรตลาด (฿39,900) แค่นิดเดียว → ตามนโยบาย **"ห้ามต่ำสุดในตลาด"** (`lib/aed/pricing.ts:188`) แต่ยังเป็นราคาที่ล่อคนเข้ามาได้
- ต้นทุนต่ำ ขายง่าย ปิดการขายไว → ได้ **ปริมาณ** ตามจุดยืน

### Y8 = ฿44,900 — ขั้นกลาง เก็บค่า feedback
- ขยับขึ้นจาก ฿39,999 เพื่อ **ไม่ชนราคา Y0** และ **เก็บเงินค่าฟีเจอร์ CPR feedback** (แทนที่จะแจกฟรี)
- ยังถูกกว่าราคา list ตลาด (฿78,900) มหาศาล และถูกกว่า street ปกติ
- สร้าง "บันได" ให้ลูกค้าขยับขึ้นไป Y2 ได้ไม่กระโดดไกล
- *ทางเลือก:* ถ้าไม่อยากแตะรุ่นที่ขายดีอยู่ คงไว้ ฿39,999 ได้ (แต่จะเสียโอกาสเก็บพรีเมียม feedback)

### Y2 = ฿59,000 — รุ่นเรือธงที่ดัน
- จุดขายเด็ด: **จอสีแสดงคุณภาพการทำ CPR สด ๆ** — ความเร็ว + ความลึก + การปล่อยสุด (full recoil)
  - ระบบเสียงอย่างเดียว "กว่าจะรู้ว่าปั๊มดี/ไม่ดี ก็ช้าไปแล้ว" — แต่เห็นบนจอ = **ปรับท่าปั๊มได้ทันที**
  - full recoil สำคัญมากต่อการคืนเลือดเข้าหัวใจ (เลือดเข้าหัวใจตอนปล่อยสุด) → ประสิทธิภาพการกู้ชีพสูงกว่าปกติมาก
- ห่างจากตัวเริ่มต้น (Y0) ~฿19,000 = ช่องอัปเซลชัดเจนขึ้น แต่ยัง **ถูกกว่าคู่แข่งรุ่นมีจอครึ่งหนึ่ง** (ZOLL/Mindray 72,500–89,000)
- ราคาป้ายขีดฆ่า (MSRP) ตั้งไว้ ฿89,000 เพื่อโชว์ส่วนลดเทียบคู่แข่ง

## 📣 แผนการตลาดที่น่าทำ (เรียงตาม ROI)

### P0 — ทำก่อน
1. **บันได Good → Best (ล่อ Y0 → ดัน Y2)**
   โฆษณา *"AED พร้อมใช้ ฿39,999"* (Y0) ดึงทราฟฟิกทุกช่อง (Google / Meta / LINE) แล้วปิดการขายด้วย *"เพิ่มจอ EKG ดู CPR สด ๆ = ฿59,000 (Yuwell Y2)"*
2. **คอนเทนต์ชูจอ EKG ของ Y2 เป็นเรือธง**
   วิดีโอ/รูปจอแสดง CPR real-time + ข้อความ *"จอ EKG ดูคุณภาพการปั๊มสด ๆ ในราคาครึ่งเดียวของ ZOLL/Mindray"* → สร้าง halo ให้ทั้งแบรนด์ แม้ยอดจริงมาจาก Y0/Y8

### P1 — ขยายปริมาณ
3. **เจาะ B2B ยกล็อต** (โรงงาน / โรงเรียน / คอนโด / ฟิตเนส / อบต.–เทศบาล) ด้วยราคาขั้นบันไดตามจำนวน — ระบบราคา tier ≥5 มีอยู่แล้วใน `calculatePrice` (`lib/aed/pricing.ts:132`)
4. **บันเดิลเพิ่มมูลค่าออเดอร์** — AED + ตู้แขวนผนัง + แผ่นสำรอง + คอร์สอบรม CPR
5. **เช่า/ผ่อน ลดกำแพงราคา** — ต่อยอดระบบใน `lib/aed/subscription.ts` เพื่อดันเครื่อง (โดยเฉพาะ Y2) ออกตลาดเป็นปริมาณ

### P2 — เสริมความเชื่อมั่น
6. **ตอกย้ำ trust badge** — อย. 65-2-2-2-0013415 · ISO 13485 · CE · รับประกัน 5–6 ปี (`primedicCertifications`)
