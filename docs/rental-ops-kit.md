# ชุดเริ่มต้นปล่อยเช่า AED (Rental Ops Kit)

> ก๊อปไปใช้ได้ทันที: สัญญาเช่า · ใบรับมัดจำ · เช็กลิสต์ตรวจเครื่อง · ทะเบียนเครื่อง · สคริปต์ปิดการขายแต่ละกลุ่ม
> ราคาอ้างอิง: รายปี 22,000 · รายเดือน 2,800 (ขั้นต่ำ 3 ด.) · รายวัน 1,500 (วันถัดไป 900) — ยังไม่รวม VAT

---

## ✅ เช็กลิสต์ก่อนเปิดรับลูกค้า (ทำครั้งเดียว)
- [ ] ซื้อเครื่องแรก 1–2 เครื่อง (เครื่องสาธิต = เครื่องปล่อยเช่า)
- [ ] ปริ้นสัญญา + ใบรับมัดจำ (ด้านล่าง) เตรียมไว้เป็นเล่ม
- [ ] ทำทะเบียนเครื่องใน Google Sheet (สคีมาด้านล่าง)
- [ ] ถ่ายรูปเครื่อง 4 ด้าน + เลขซีเรียล เก็บเป็น "สภาพตั้งต้น"
- [ ] เตรียมกล่อง/กระเป๋าขนส่ง + สติกเกอร์ติดเลขเครื่อง
- [ ] ตั้งเตือนวันหมดอายุแผ่น/แบตในปฏิทิน

## 🎯 5) สคริปต์ปิดการขาย แยกตามกลุ่ม

### 🏃 อีเวนต์ / งานวิ่ง-กีฬา (ปิดเร็วสุด — เริ่มที่นี่)
> "งานของพี่มีคนเป็นร้อย/พัน เกิดหัวใจหยุดเต้นมีเวลาแค่ 4 นาที เรามี AED ให้เช่ารายวัน 1,500 ส่ง-รับถึงงาน พร้อมแนะนำการใช้หน้างาน จองวันไหนครับ?"
- หา: เพจจัดงานวิ่ง/มาราธอน, ออร์แกไนเซอร์, สนามกีฬา, ฟิตเนสที่จัดอีเวนต์
- ปิดด้วย: เก็บมัดจำ + ค่าเช่าล่วงหน้า, ยืนยันวันส่ง-รับ

### 🏢 ออฟฟิศ / โรงงาน / ฟิตเนส (รายปี = คืนทุน)
> "ตามกฎความปลอดภัย สถานประกอบการควรมี AED ซื้อขาด 3-4 หมื่น แต่เช่ารายปีแค่ 22,000 รวมติดตั้ง+อบรม+ดูแล ออกใบกำกับภาษีได้ ขอส่งใบเสนอราคาไหมครับ?"
- หา: HR/จป. (เจ้าหน้าที่ความปลอดภัย), ผู้จัดการอาคาร, นิติบุคคลคอนโด
- ปิดด้วย: เสนอ PO ยกเว้นมัดจำได้, ชูเปลี่ยนแผ่นฟรีถ้าใช้จริง

### 🔄 รายเดือน (สำหรับคนยังไม่กล้าผูกปี)
> "ลองเช่ารายเดือนก่อนก็ได้ครับ 2,800/เดือน ถ้าใช้แล้วโอเค อยากทำรายปี ค่าเช่าที่จ่ายมาหักเป็นส่วนลดให้เลย"

## 📌 ลำดับลงมือ (สรุป)
1. ซื้อ 1–2 เครื่อง → 2. เตรียมสัญญา/เช็กลิสต์/ชีต → 3. โพสต์ + ยิงแอดเจาะอีเวนต์ & องค์กร
4. ปิดรายปี 1 ดีล = คืนทุนเครื่องแรก → 5. เครื่องที่ 2 รับอีเวนต์เสาร์-อาทิตย์
6. จองเกิน 70% เมื่อไหร่ → ค่อยซื้อเครื่องเพิ่ม
