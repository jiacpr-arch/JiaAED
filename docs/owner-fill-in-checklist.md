## 3) ตัดสินใจเรื่องราคาเช่าซื้อ / เช่าบริการ  💰

โค้ดมี TODO ค้างเพราะราคาขัดกันเอง ต้องเจ้าของยืนยันตัวเลขที่ถูกต้อง:

### 3.1 Safety Start (เช่าแล้วได้ซื้อ) — `lib/aed/packages.ts:57`
- ปัจจุบัน: มัดจำ ฿15,000 + ผ่อน ฿2,500 × 18 เดือน = **฿60,000**
- ปัญหา: น้อยกว่าซื้อสด Premium (฿69,900) → คนจะเลือกผ่อนแทนซื้อสด
- ต้องทำ: ยืนยันว่าตั้งใจให้เช่าซื้อถูกกว่าซื้อสดจริงไหม หรือปรับยอดผ่อน/จำนวนงวด

### 3.2 Safety Care (เช่าบริการครบวงจร) — `lib/aed/packages.ts:80`
- ปัจจุบันบนเว็บ: เริ่ม ฿1,990/เดือน (อิงตาม subscription tier BASIC)
- ปัญหา: เอกสารเก่าบางที่เขียน "฿1,500/เดือน" → ขัดกัน
- ต้องทำ: เลือกราคาเดียวที่ถูกต้อง

### 3.3 ราคาพื้นตลาดของอะไหล่ (แผ่น Pad / แบต) — `lib/aed/pricing.ts:187`
- ใช้ตรวจไม่ให้เราตั้งราคาต่ำกว่าตลาด (มี unit test คุมอยู่)
- ปัจจุบันเดาไว้: pad ฿4,000 / battery ฿6,000
- ต้องทำ: ยืนยันราคาคู่แข่งจริงที่เห็นในตลาด (ถ้าเปลี่ยน ต้องไม่ให้ minPrice ของเรา ≤ ค่านี้ ไม่งั้น test จะ fail เตือน)

## 4) ทดสอบ flow ออกใบเสนอราคา + ลิงก์ชำระเงิน (ต้องมี API key จริง)  🔌

โค้ดพร้อมแล้ว แต่ผมทดสอบ end-to-end ไม่ได้เพราะไม่มี key จริง เจ้าของช่วยลองจริง 1 รอบ:
- ส่งข้อความเข้า LINE OA → ขอใบเสนอราคา → เช็กว่า FlowAccount ออกเอกสารจริง (`FLOWACCOUNT_*` env)
- เช็กว่า Stripe payment link สร้างได้จริง (`STRIPE_SECRET_KEY` env)
- ถ้าพังจะมี VIP fallback (บอทไม่บอกลูกค้าว่า error + แจ้งเจ้าของให้ทำมือ) — แต่ควรให้ของจริงทำงาน
